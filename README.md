## ToDo App REST API  

Приложение REST API для управления повседневными задачами.  
Возможности API:
1. Профиль пользователя:
    - Регистрация
    - Вход в профиль
    - Выход из профиля
    - Редактирование профиля
2. Списки задач ***(todo-list)***
3. Задачи к определённому списку ***(todo-item)***
